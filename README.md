# ProfileDeck

ProfileDeck is a sleek, modern web application that allows you to view user profiles in a responsive card layout. Each card displays key user information including avatar, name, email, gender, age, and location. The app fetches data dynamically from a public API and includes interactive features such as a refresh button with smooth loading animations. Built with TailwindCSS, ProfileDeck comes with a stylish dark mode interface.

## Features

* Responsive 3-column card layout that adapts to mobile, tablet, and desktop screens.
* Dark mode design using TailwindCSS.
* Dynamic fetching of user data from [Random User API](https://randomuser.me/).
* Refresh button with smooth spinning animation for the loading state.
* Hover animations on cards for better interactivity.
* Displays detailed user information: avatar, name, email, gender, age, location.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/ProfileDeck.git
```

2. Navigate into the project directory:

```bash
cd ProfileDeck
```

3. Open `index.html` in your favorite browser.

> No backend required; the app uses the public Random User API.

## Usage

* Click the **Refresh** button to load new user profiles.
* Hover over cards to see the interactive animations.
* Works seamlessly across different screen sizes.
